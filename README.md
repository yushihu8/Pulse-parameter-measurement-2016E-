# Pulse Parameter Measurement 2016E

基于 `STM32F407VET6 + FPGA` 的脉冲信号参数测量仪工程，对应 2016 年 TI 杯大学生电子设计竞赛 E 题。项目面向脉冲信号的高速采样与参数测量，当前实现了幅值、上升时间、频率、占空比测量，以及 LCD 实时显示、串口波形输出和双通道量程切换。

## 项目简介

本项目采用“FPGA 负责高速采样与硬件计数，STM32 负责控制、计算与显示”的协同架构：

- FPGA 端完成 ADC 采样、缓存管理、SPI 寄存器接口、频率/占空比硬件测量
- STM32 端完成数据读取、波形参数分析、按键交互、LCD 显示和串口输出
- 系统支持直通通道与衰减通道切换，适配不同幅度范围的输入脉冲信号

## 主要功能

- 幅值测量
- 上升时间测量
- 频率测量
- 占空比测量
- LCD 实时显示测量结果
- 串口输出 ADC 原始波形
- 双测幅通道切换
- 大信号场景下自动切换到衰减通道

## 测量模式

### `Wave` 模式

用于测量：

- 脉冲幅值 `Amp(Vpp)`
- 上升时间 `Rise(ns)`

### `Freq` 模式

用于测量：

- 频率 `Freq`
- 占空比 `Duty`

## 通道说明

- `CH1 0dB`
  - 直通测量通道
  - 适合中小幅度信号
- `CH2 -10dB`
  - 经 10 dB 衰减器后的测量通道
  - 适合较大幅度信号，避免 FPGA/ADC 采样打满

## 按键功能

- `KEY2`：输出当前采样波形到串口
- `KEY3`：切换 `Wave / Freq` 测量模式
- `KEY4`：切换测幅通道 `CH1 / CH2`

## 关键实现思路

### FPGA 部分

- `ad9248.v`：ADC 采样接口
- `adc_ram_control.v`：采样数据缓存控制
- `freq_duty_meter.v`：频率与占空比硬件测量
- `spi_slave.v / spi_reg_control.v / reg_control.v`：与 STM32 的 SPI 寄存器通信
- `top.v`：系统顶层连接

### STM32 部分

- 通过 SPI 读取 FPGA 采样缓冲区
- 对波形高低电平进行估计
- 计算 10% / 90% 阈值交点
- 插值计算上升时间
- 读取 FPGA 返回的硬件计数值并换算频率、占空比
- 在 LCD 上持续刷新显示结果

## 标定与微调

项目中已经预留了幅值与时间系数修正能力，可用于后续标定：

- 幅值通道系数可分别对 `CH1`、`CH2` 调整
- 幅值全局系数可统一修正整体偏差
- 上升时间换算可进一步加入通道级修正系数
